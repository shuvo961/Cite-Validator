Supported formats

Paste almost any bibliography text

Cite Validator supports common styles and messy copied references from PDFs. The app detects citation style automatically when possible.

Academic styles

APA, IEEE, MLA, Chicago, Vancouver, journal-style references

Identifiers

DOI, ISBN, ISSN, PMID, PMCID, arXiv ID, URLs

Imports

TXT, BibTeX, RIS, CSV, copied bibliography sections

Messy PDF text

Broken lines, page numbers, broken DOI URLs, numbered bibliographies, pasted-together references
